# QR Router API v1.0.0

Cloudflare Workers + D1 + KV
*(🔒 Requiere JWT)*
Route: https://qr-container.pedrocarpiom.workers.dev/

## Auth

* **`POST /auth/signup`** | Body: `email`, `password` (min 8) | Res: 200 (`access_token`, `refresh_token`), 409
* **`POST /auth/login`** | Body: `email`, `password` | Res: 200 (`access_token` 15 min, `refresh_token` 7 días), 401
* **`POST /auth/refresh`** | Body: `refresh_token` | Res: 200 (`access_token`), 401

## Onboarding

* **`POST /api/company`** 🔒 | Body: `company_name`, `slug` (`^[a-z0-9-]+$`) | Res: 200, 409

## QRs

* **`POST /api/qrs/fallback`** 🔒 | Body: `qr_string`, `expiration_date`, `bank` (opt) | Res: 200, 400
* **`POST /api/qrs`** 🔒 | Body: `amount` (>0), `qr_string`, `expiration_date`, `bank` (opt) | Res: 200
* **`GET /api/qrs`** 🔒 | Query: `page` (def 1), `limit` (def 20, max 100) | Res: 200 (paginado)
* **`GET /api/qrs/expiring`** 🔒 | Res: 200 (lista <30 días)

## Admin 🔒 *(x-internal)*

* **`POST /api/approve`** | Body: `user_id` (UUID) | Res: 200, 403, 404
* **`GET /api/admin/stats`** | Res: 200, 403
* **`GET /api/admin/pending`** | Query: `page`, `limit` | Res: 200, 403
* **`GET /api/admin/users`** | Query: `page`, `limit`, `status` | Res: 200, 403
* **`GET /api/admin/users/{user_id}/qrs`** | Path: `user_id` | Query: `page`, `limit` | Res: 200, 403, 404

## Router

* **`GET /router/{slug}/{amount}`** | Path: `slug`, `amount` (>0) | Res: 200 (`qr_string`), 404, 429
